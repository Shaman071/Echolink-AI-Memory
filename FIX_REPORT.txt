KNOWN LIMITATIONS:
================================================================================

1. Vector Search:
   - Current implementation uses in-memory cosine similarity
   - Suitable for < 10,000 fragments
   - For production, use vector DB (Pinecone, Weaviate)

2. Embedding Service:
   - Uses sentence-transformers (384-dim vectors)
   - Falls back to local if OpenAI key not configured
   - Max embedding time: ~5 seconds per fragment

3. Graph Rendering:
   - Force-graph uses D3 force simulation
   - Performance degrades with > 500 nodes
   - Consider simplifying graph for large datasets

4. File Upload:
   - Max size: 50MB
   - Supported: TXT, PDF, DOCX, PNG, JPG
   - OCR for images uses Tesseract (requires installation)
